# Sistema Financeiro Pessoal (EM DESENVOLVIMENTO).

Sistema financeiro pessoal.

O intuito desse sistema é aplicar conhecimentos de arquitetura e implementações para aprender como está o meu conhecimento em Laravel.

## Objetivo do projeto

- Projeto de controle financeiro pessoal, onde vamos ter lançamentos de contas a receber, contas a pagar separados por carteiras e categorias.
- Envio de e-mail notificando as contas que estão para vencer.
- Relatório de contas pagas/receber por período, categoria e fornecedores.
- Relatório de provisões futuras.

## 1 - Arquitetura e Padrões

O projeto vai estar utilizando o padrão do Laravel para aplicação da sua tecnologia, como request, service provider, resources e a camada de service da forma correta.

## 2 - O projeto vai ser evoluido para utilização de novas tecnologias e arquiteturas como:

- Aplicar filas para envio de e-mail de ativação da conta.
- Implementação de testes unitários e end-to-end.
- Aplicação de repositories, DDD e Clean Arch.

## 3 - Tecnologias.

###### Backend:
- Laravel 12

###### Fronend:
- NuxtJs

###### DB:
- Sqlite (inicialmente)
- Postgres (futuramente)

###### IA:
- Integração com o N8N para envio de mensagens e interação com a IA.

**Nota:** Esse projeto é um projeto de estudo para validar o que eu tenho de conhecimento relacionado ao Laravel e o quanto eu posso trabalhar a melhoria dele.
